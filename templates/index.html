// Capabilities

What It Detects

SQL Injection
Detects error-based and time-based blind SQL injection across all forms and URL parameters automatically.
CRITICAL
📜
Cross-Site Scripting
Tests reflected XSS by injecting payloads into every input and checking if they appear unescaped in the response.
HIGH
🛡️

MEDIUM
📂
Sensitive Files

CRITICAL
🔁

MEDIUM
🕷️

// Process

How It Works

01
Target Input
Enter the target URL. Rebel Protocol accepts any http or https web address and optional scan depth settings.
02
Crawl & Map
The crawler visits every page, follows links, and extracts all forms, input fields, and URL parameters.
03
Inject & Test
50+ attack payloads are injected into each discovered input. Server responses are analyzed for vulnerability signatures.
04
Report Generation
All findings are sorted by severity — CRITICAL to LOW — and presented in a structured, actionable report.
